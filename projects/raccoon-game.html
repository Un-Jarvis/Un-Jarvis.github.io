Running Raccoon Game

Description

  • A game created using Processing.
  • The game has an splash screen with instructions, three difficulty levels, reward systems, and a score list board.
  • In the game, players control the Raccoon character to dodge the obstacles and collect coins to gain points.
  • All game assets were purchased from Bevouliin.
