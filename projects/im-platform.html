Instant Messaging Platform

Description

  • Project in Computer Networking Course (CSE 3461) directed by David Ogle.
  • Created a platform that supports both broadcast and direct messages among multiple users. Clients are able to see other online clients.
  • The server registers each client with a unique username and password, handles multiple connections from multiple clients, and keeps track of all client information in a database.
  • Implemented a graphical user interface for the client side which connects to the back-end application. Users can type their messages through the GUI.
  • Used model-view-controller (MVC) architecture.
